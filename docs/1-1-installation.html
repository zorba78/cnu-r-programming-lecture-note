1.1 R 설치하기

R 다운로드 사이트: https://www.r-project.org 또는 https://cran.r-project.org

  1. 웹 브라우저(i.e. Explore, Chrome, Firefox 등)의 주소 입력창에 https://www.r-project.org

  2. 좌측 R Logo 하단 Download 아래 CRAN 클릭

  1. 클릭 후 연결한 페이지를 스크롤 후 Korea 아래 링크1 클릭

  1. 클릭 후 세 가지 운영체제(Linux, Mac OS X, Windowns)에 따른 R 버전 선택 가능2

  1. Downloads R for Windows 링크 클릭하면 다음과 같은 화면으로 이동

  1. 위 화면에서 base 링크 클릭 후 아래 화면에서 Downloads R 3.4.2 for Windows 를 클릭 후 설치 파일을 임의의 디렉토리에 저장 및 실행

  1. 해당 링크들은 접속 시점에 따라 변경될 수 있음↩

  2. 본 노트는 Windows 버전 설치만 다룸↩
